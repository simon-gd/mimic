Comparación de 2 proporciones (Binomial) aproximadas a una Normal

X: Variable aleatoria continua. Y: Variable aleatoria discreta.

Se desea contrastrar la diferencia de proporciones (Binomial) de 2 poblaciones: p1 y p2 usando sus valores experimentales, contra una distribución Normal tipificada.

La aproximación a la Binomial será válida cuando se satisfacen las condiciones:
n ≥ 20, n*p ≥ 5, n*q ≥ 5. Con q = 1-p0.
