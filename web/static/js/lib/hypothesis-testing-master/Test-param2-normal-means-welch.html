Sistema Estadístico de Contraste de Hipótesis



Comparación de 2 medias (μ) en variables Normales independientes (Welch)

X: Variable aleatoria continua. Y: Variable aleatoria continua.

Se desea contrastrar la media de 2 poblaciones: μ1 y μ2, con las varianzas poblacionales σx2 σy2 desconocidas. Se utiliza un estimador para las varianzas. El constraste se realiza comparando los valores experimentales contra una distribución t-Student.

Primero se require que las varianzas sean diferentes. Comprobar por una Comparación de Varianzas.

El caso de test unilaterales, se require:
Si H1 ≡ (μ1 > μ2) => X̄ > Ȳ
Si H1 ≡ (μ1 < μ2) => X̄ < Ȳ
Esto implica que podría ser necesario intercambiar los valores de X por los de Y.

El cálculo del Valor P se realiza con una interpolación lineal entre 2 resultados aproximados. Es posible que la región de aceptación del contraste difiera levemente de la comparación: Valor P < α. La interpretación del resultado se realiza con el Valor P.
